

Build a **high-performance, SEO-optimized, mobile-friendly “All-in-One PDF Tools Website”** that is suitable for Google AdSense approval and global organic traffic.

---

## 🧾 Website Concept:

Create a modern SaaS-style website named:
**“Smart PDF Tools”** (or any brandable name)

It should provide free online PDF utilities for users worldwide.

---

## 🧰 Core Tools (Must Include):

Each tool must have its own section/page:

* 📄 PDF to Word Converter
* 📝 Word to PDF Converter
* 🔗 Merge PDF Tool
* 📉 Compress PDF Tool
* 🖼️ Image to PDF Tool

---

## 🎨 UI / UX Requirements:

* Clean, modern SaaS-style design
* Fully responsive (mobile + tablet + desktop)
* Fast loading (under 2–3 seconds)
* Minimal animations (no heavy JS libraries)
* Tool cards with icons and hover effects
* Drag & drop file upload UI
* Progress bar for processing simulation
* Dark mode toggle (optional but recommended)
* Sticky navigation bar

---

## 🔍 FULL SEO REQUIREMENTS (VERY IMPORTANT):

### 🧠 On-Page SEO:

* Unique **meta title for every page**
* Optimized meta description (150–160 chars)
* Proper keyword usage (PDF tools, convert PDF, compress PDF, etc.)
* One H1 tag per page only
* Structured H2/H3 headings
* Internal linking between all tools
* SEO-friendly URLs:

  * /pdf-to-word
  * /merge-pdf
  * /compress-pdf

---

### ⚡ Technical SEO:

* Lightweight HTML/CSS/JS (no unnecessary libraries)
* Fast page speed (Google PageSpeed 90+ target)
* Lazy loading images
* Minified CSS/JS
* Proper sitemap.xml
* robots.txt file included
* Canonical tags on all pages

---

### 📱 Mobile SEO:

* Fully responsive layout
* Touch-friendly buttons
* No horizontal scrolling
* Mobile-first design approach

---

## 📄 Required Pages (For AdSense Approval):

Must include:

1. Home Page (overview of all tools)
2. Individual Tool Pages (each tool with description)
3. About Us Page (trust building)
4. Contact Page (working form)
5. Privacy Policy Page (very important for AdSense)
6. Terms & Conditions Page
7. Disclaimer Page (optional but recommended)

---

## 🧾 Content Requirements (VERY IMPORTANT FOR SEO):

Each tool page must include:

* 300–600 words unique SEO description
* How-to-use section
* Benefits of tool
* FAQ section (3–5 questions)
* Keywords naturally placed (no keyword stuffing)
